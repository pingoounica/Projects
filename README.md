# Projects📋 Gerenciador de Tarefas em Python
🚀 Sobre o projeto

Este projeto é um gerenciador de tarefas em Python via terminal, desenvolvido com o objetivo de praticar lógica de programação, manipulação de dados e persistência em arquivos.

A aplicação permite ao usuário organizar tarefas de forma simples e eficiente, simulando funcionalidades básicas de sistemas reais.

⚙️ Funcionalidades

✔️ Adicionar tarefas
✔️ Listar tarefas
✔️ Marcar como concluída
✔️ Remover tarefas
✔️ Salvamento automático em arquivo .json

🛠️ Tecnologias utilizadas
🐍 Python
📁 JSON (armazenamento de dados)

▶️ Como executar
# Clone o repositório
git clone https://github.com/pingoounica/Projects.git

# Acesse a pasta
cd Projects

# Execute o projeto
python tarefas.py
